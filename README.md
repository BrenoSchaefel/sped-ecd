# SPED ECD

Biblioteca Java para leitura, validação e geração de arquivos do SPED ECD (Escrituração Contábil Digital).

## Requisitos

* Java 17+
* Maven 3.8+

## Instalação

Adicione a dependência ao seu `pom.xml` (após a publicação no Maven Central):

```xml
<dependency>
    <groupId>io.github.brenoschaefel</groupId>
    <artifactId>sped-ecd</artifactId>
    <version>1.0.0</version>
</dependency>
```
